## 教程   
1.自行学习其他老项目的抓包教程（有时间复制过来）  
2.将认证服务器ip，mac地址，替换到文件（忘记用ai添加这个参数了，同学校的不用自己编译)  
3.编译（等我慢慢写）  
4.运行 ./文件名 -u username -p password -i 你认证前获取的ip或者你抓包得到的hostip （选填-v，开启log输出）
